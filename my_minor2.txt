Program Code
292
Program Notes

No grade less than C- in Minor courses.
Program Course Requirements

Core
8
Total Credits
Complete all of the following
Take 1 of the following: 
CSE110 - Introduction to Programming (2)
PH135 - Introduction to Scientific Computing in Physics (2)

AND
Take the following: 
CSE111 - Programming with Functions (2)
CSE210 - Programming with Classes (2)
CSE212 - Programming with Data Structures (2)

MATH119 - Applied Calculus for Data Analysis (4)

Take the following: 
MATH341 - Linear Algebra (3)
